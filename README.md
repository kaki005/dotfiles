# dotfiles

## Usage
```
git clone https://github.com/kaki005/dotfiles.git
dotfiles/scripts/link.sh
dotfiles/scripts/install.sh 
```
## Keybinding
- `Ctrl + Q`
  - 過去に移動したDirectoryを選んで移動
- `Ctrl + R`
  - 過去に実行したコマンドから選んで実行
- `Ctrl + ]`
  - gitレポジトリへ移動
- `Ctrl + W`
  - 小ディレクトリから選んで移動
